PreçoRadar

PreçoRadar is an application that tracks the daily prices of essential products in supermarkets across Belo Horizonte. Fully developed by me, the project automates real-time data collection and displays everything in a clear and interactive way for users.

Aimed at solving the difficulty of tracking price changes amid scattered or manipulated information, the app delivers a simple and intuitive experience on both mobile and desktop devices.

  • Automated scraping of APIs and websites from supermarkets in Belo Horizonte
  • Frontend built with React and interactive data visualizations (Recharts)
  • Node.js + Express backend with robust data processing logic
  • PostgreSQL database managed with Prisma ORM

Orion-NDK

Orion-NDK is a mini social network inspired by the biggest names in the industry. Developed by me and based on modern platforms used worldwide, the project allows people to express themselves and share their daily lives in a fast and intuitive way.

Users can post photos, create and edit their profiles, add friends and acquaintances, and interact with each other's posts. With real-time notifications and a responsive design, the experience is seamless across both desktop and mobile devices.

  • Responsive and intuitive frontend built with React.
  • Real-time notifications via WebSockets (Socket.IO).
  • Node.js + Express backend with REST architecture.
  • PostgreSQL database managed with Prisma ORM.

PreçoRadar

PreçoRadar is a full-stack application that tracks the daily prices of essential items in supermarkets around Belo Horizonte. Built entirely by me, the project automates data collection and displays it in a clear, accessible way.

  • Automated scraping of external supermarket APIs
  • Frontend in React with interactive charts (Recharts)
  • Node.js + Express backend
  • PostgreSQL database with Prisma ORM

OrionNDK

OrionNDK is an experimental social network built from scratch, featuring real-time functionality. Inspired by modern social platforms, the project demonstrates the integration of a reactive frontend, scalable backend, and secure authentication.

  • Responsive React frontend
  • RESTful backend with Express + Node.js
  • Real-time updates with WebSockets
  • Secure authentication with JWT
  • PostgreSQL database with Prisma ORM

PreçoRadar

PreçoRadar é uma aplicação fullstack que rastreia diariamente os preços de itens essenciais em supermercados de Belo Horizonte. Desenvolvido 100% por mim, o projeto automatiza a coleta e exibição de dados de forma clara e acessível.

  • Scraping automatizado de APIs externas de supermercados de BH
  • Frontend em React com gráficos interativos (Recharts)
  • Backend Node.js + Express
  • Banco de dados PostgreSQL com Prisma ORM

OrionNDK

OrionNDK é uma rede social experimental, construída do zero, com funcionalidades em tempo real. Inspirado nas redes modernas, o projeto demonstra integração entre frontend reativo, backend escalável e autenticação segura.

  • Frontend em React com design responsivo
  • Backend RESTful em Express + Node.js
  • WebSockets para atualizações em tempo real
  • Autenticação segura com JWT
  • Banco de dados PostgreSQL com Prisma ORM

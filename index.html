PreçoRadar

PreçoRadar é uma aplicação que rastreia diariamente os preços de produtos essenciais em supermercados de Belo Horizonte. Desenvolvido inteiramente por mim, o projeto automatiza a coleta de dados em tempo real, e exibe tudo de forma clara e interativa para o usuário.

Focado em acabar com a dificuldade de acompanhar variações de preço em meio a informações dispersas ou manipuladas, a aplicação entrega uma experiência simples e intuitiva, em dispositivos móveis e desktop.

  • Scraping automatizado de APIs e sites de supermercados de BH
  • Frontend em React com visualizações interativas (Recharts)
  • Backend Node.js + Express com lógica robusta de manipulação de dados
  • Banco de dados PostgreSQL gerenciado com Prisma ORM
